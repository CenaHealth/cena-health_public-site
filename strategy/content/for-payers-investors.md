    - *System-economic alignment with VBC contracts* — the reporting work the substrate is being built to do automatically (what currently lives in half-FTE-of-spreadsheets).
    - *(Voice owner's call: include 3–4 of the above; 5 is a stretch list. The criterion is "which primitive does the payer/investor reader most recognize as costly + currently unsolved?")*
  - Lead each primitive with the partner / care team / program / payer as the grammatical subject; Cena's substrate is the structural object (`Partner-First`).
  - Hold the brownfield posture from the next section in voice — even though the explicit brownfield framing lands in Section 3, the value-being-built section should not read as a greenfield pitch.
- **Must-not:**
  - Make any present-tense Cena capability claim. **The single highest-risk failure mode on this page.** Cap-Honest gate ledger Hotspot 2 cross-step pattern: *"Sentences that respect the bifurcation can be HONEST-TODAY on either half"* — but this section names *Cena's value*, where the partner/Cena bifurcation collapses into one subject (Cena).   - Frame the substrate as *AI-powered* / *AI-driven* — voice.md §4 internal-vocab translation requires "Cena's clinical intelligence platform" / "the substrate" instead.

- **Voice cues:**
  - **The tense discipline.** Every Cena-as-subject sentence must pass the *"is this what's running today, or what's being built?"* test. If today, the bifurcation discipline says reroute to the partner as subject; if being built, the verb is honest future-tense. There is no third option on this page.
  - **The value frame's character.** Each primitive should read as something a payer medical director or VBC CFO recognizes as *missing today* and *worth building*.
